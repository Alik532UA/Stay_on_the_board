# 🔘 План уніфікації стилів кнопок (Button Unification Plan)

**Мета:** Замінити розрізнені реалізації кнопок (`<button class="...">`) на єдиний компонент `StyledButton.svelte` без візуальних змін для користувача.

**Стратегія:** "Надмаленькі кроки". Ми не ламаємо існуючий CSS, а створюємо новий компонент, перевіряємо його, і тільки потім точково замінюємо використання.

## 📋 Аналіз поточного стану

Ми маємо декілька "типів" кнопок:
1.  **Generic Modal Button** (`modal.css` -> `.modal-btn-generic`):
    *   Розмір: `padding: 12px 26px`, `font-size: 1.1em`.
    *   Варіанти: Default (White), Primary (`green-btn`/`primary`), Info (`blue-btn`), Danger (`danger-btn`).
2.  **Main Menu Button** (`main-menu.css` -> `.modal-button`):
    *   Розмір: Великі (`padding: 24px 32px`, `font-size: 1.2em`).
    *   Варіанти: Secondary (Default), Danger.
3.  **Toggle Button** (`ToggleButton.svelte`):
    *   Кастомні стилі, залежать від `variables.css`.

## 🧪 План верифікації
Для кожного кроку міграції виконувати:
1.  `npm run dev`
2.  Відкрити відповідну сторінку/модалку.
3.  Порівняти візуально (можна зробити скріншот "До").

5.  Натиснути хоткей (якщо є) - перевірити спрацювання.
