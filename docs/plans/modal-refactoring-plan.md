## Аналіз структури модального вікна

### Типи контенту (за dataTestId)
1. **game-over-modal** — результати гри
2. **replay-modal** — перегляд реплею
3. **board-resize-confirm-modal** — підтвердження зміни розміру
4. **expert-mode-modal** — налаштування експерта
5. **rules-modal** — правила гри
6. **controls-modal** — керування
7. **faq-modal** — FAQ

### Компоненти всередині
- `SvgIcons` — іконки
- `ReplayViewer` — через `$modalState.component`
- `FinalScoreDisplay` — для game-over

├── Modal.svelte                 (Базовий каркас — overlay, header, footer)
