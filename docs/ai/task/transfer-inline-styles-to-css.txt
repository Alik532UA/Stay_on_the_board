Перенести всі inline-стилі (style="...") у відповідні CSS-файли по всьому проекту, щоб вигляд і розміри елементів керувалися тільки через CSS.

1. Знайти всі style="..." у JS/HTML.
2. Створити відповідні CSS-класи у відповідних CSS-файлах.
3. Замінити style="..." на class="..." у шаблонах компонентів.
4. Перевірити вигляд у браузері.
5. Підтримувати цю практику для всіх нових компонентів.
