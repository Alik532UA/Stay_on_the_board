Баг репорт #20: Ферзь завжди з'являється по центру дошки

Дата: 2024-07-13
Пріоритет: Середній
Статус: НЕВИПРАВЛЕНО

ОПИС ПРОБЛЕМИ:
Після старту гри ферзь завжди розміщується по центру дошки, незалежно від розміру. Це робить старт гри передбачуваним.

КРОКИ ДЛЯ ВІДТВОРЕННЯ:
1. Запустити нову гру
2. Звернути увагу на позицію ферзя

Очікуваний результат:
- Ферзь повинен з'являтися у випадковій клітинці дошки

Фактичний результат:
- Ферзь завжди з'являється по центру дошки
