Баг репорт #23: Чекбокс "Показати дошку" не працює

Дата: 2024-07-13
Пріоритет: Середній
Статус: НЕВИПРАВЛЕНО

ОПИС ПРОБЛЕМИ:
Перемикач "Показати дошку" не впливає на відображення дошки — вона завжди показується незалежно від стану чекбокса.

КРОКИ ДЛЯ ВІДТВОРЕННЯ:
1. Запустити гру
2. Перемкнути чекбокс "Показати дошку"
3. Перевірити, чи змінилося відображення дошки

Очікуваний результат:
- Дошка (і чекбокс "Показувати доступні ходи") повинна зникати/з'являтися відповідно до стану чекбокса

Фактичний результат:
при вимкнені чекбокса "Показати дошку" зникає все, а повинно лише Дошка (і чекбокс "Показувати доступні ходи")
