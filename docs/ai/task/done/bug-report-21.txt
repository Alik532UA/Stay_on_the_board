Баг репорт #21: Доступні ходи не оновлюються після ходу

Дата: 2024-07-13
Пріоритет: Середній
Статус: НЕВИПРАВЛЕНО

ОПИС ПРОБЛЕМИ:
Після виконання ходу доступні ходи не оновлюються автоматично. Це ускладнює орієнтацію для користувача.

КРОКИ ДЛЯ ВІДТВОРЕННЯ:
1. Запустити гру
2. Зробити хід
3. Перевірити, чи оновилися доступні ходи

Очікуваний результат:
- Доступні ходи повинні оновлюватися після кожного ходу користувача

Фактичний результат:
- Доступні ходи залишаються такими ж, як до ходу
