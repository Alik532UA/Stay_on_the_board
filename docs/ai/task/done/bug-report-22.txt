Баг репорт #22: Чекбокс "Показувати доступні ходи" не працює

Дата: 2024-07-13
Пріоритет: Середній
Статус: НЕВИПРАВЛЕНО

ОПИС ПРОБЛЕМИ:
Перемикач "Показувати доступні ходи" не впливає на відображення доступних ходів — вони завжди показуються незалежно від стану чекбокса.

КРОКИ ДЛЯ ВІДТВОРЕННЯ:
1. Запустити гру
2. Перемкнути чекбокс "Показувати доступні ходи"
3. Перевірити, чи змінилося відображення доступних ходів

Очікуваний результат:
- Доступні ходи повинні зникати/з'являтися відповідно до стану чекбокса

Фактичний результат:
- Доступні ходи завжди показуються, незалежно від стану чекбокса
