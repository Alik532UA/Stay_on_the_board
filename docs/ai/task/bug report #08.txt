Баг репорт #08: Показ доступних ходів не вимикається

Дата: 15.01.2025
Пріоритет: Високий
Статус: Відкритий

ОПИС ПРОБЛЕМИ:
Коли користувач вимикає налаштування "Показувати доступні ходи" через чекбокс, доступні ходи все одно продовжують з'являтися на дошці.

КРОКИ ДЛЯ ВІДТВОРЕННЯ:
1. Запустити гру з комп'ютером
2. Переконатися, що налаштування "Показувати доступні ходи" увімкнено
3. Побачити, що доступні ходи відображаються на дошці (крапки)
4. Вимикнути чекбокс "Показувати доступні ходи"
5. Зробити хід
Очікуваний результат: доступні ходи не показують
Фактичний результат: доступні ходи залишаються видимими
